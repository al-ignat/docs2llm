# Changelog

## v0.1.0 — Initial Release

- 6 commands: Convert File, Convert Clipboard, Quick Convert, Smart Copy, Smart Paste, Smart Save
- Bidirectional conversion: auto-detects inbound (to Markdown) or outbound (to Word/PowerPoint/HTML)
- Smart auto-detection: Finder selection, text selection, clipboard content
- Engine and quality metadata in result view (engine name, quality score, OCR status)
- Tesseract error detection with actionable install hint
- Per-command format preferences (inbound format, export format)
- Template support for outbound conversion via Pandoc
